Welcome

Titan Code Analysis is a project that aims to build an LLM vulnerability scanner that can take a function and label it as: vulnerable or not-vulnerable. It is a research project conducted by the Singapore Management University, collaborating with GovTech. The project is funded by the Smart Nation Group.
