Open-Source Tool Ecosystem

The TITAN project has developed and released several open-source tools for vulnerability detection, code analysis, and security assessment.

Vulnerability Detection & Analysis

SecTor
VFC (Vulnerability-Fixing Commit) classification for Java commits
PatchSeeker
Detecting vulnerability-fixing commits from NVD and GitHub
VulSifter
Detecting vulnerable changes in a vulnerability-fixing commit
InSA
Assessing vulnerability severity
VulCoCo
Vulnerability detection using code clone analysis
R2Vul
Vulnerability detection with reinforcement learning and structured reasoning
VulTrial
Vulnerability detection using multi-agent debate system

Code Synthesis

RVG
Vulnerable code synthesis for security testing
